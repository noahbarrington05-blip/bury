Drop your custom fonts in this folder.

Recommended setup
- first-name.woff2
- last-name.woff2

Other formats can work too
- .woff
- .ttf
- .otf

Best option
- Use .woff2 if you have it.

When the files are here, tell Copilot which font should be used for first names and which should be used for surnames, and the editor can be updated to load them with @font-face.
